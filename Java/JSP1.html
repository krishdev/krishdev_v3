What is JSP?
Java code embedded in HTML
Lifecycle methods of JSP
jspInit, _jspService, jspDestroy
JSP life cycle
JSP -> translator (JSP Engine) -> .java file -> compiler -> .class file -> loaded in to webcontainer as a servlet. This is during first time execution or after a change in JSP file
Otherwise servlet is loaded directly from the web container. So JSP is like any other servlet once its loaded.
Different JSP Element
Directives, Declarations, Scriplets, Expressions
What are JSP Declarations and why we need them?
Declarations are within <%! %> or <jsp:declaration> </jsp:declaration> tags
All variable and method declarations are declared within this block. It goes outside the service method, its something like instance variable and members
variables declared like this can be shared between multiple requests. and variable thats used across the jsp page can be declared inside <%! %>
What are JSP Expressions and why we need them?
Code inside <%= %> are called expressions
Code inside the expression element are evaluated
Can be used when we want to print some value to the console or to assign a value to some other variable
What are JSP Scriplets and why we need them?
Scriplets are within <% %> or <jsp:scriplet> </jsp:scriplet> tags
Java code can be added inside scriplets. This is used to execute small logic. if the logic is complex and large its better to be added in the servlet than in jsp.
What are JSP Directives and why we need them?
Directives are within <%@ %> tags
There are different types of directives -> page, include, tablib
<%@ page import="java.util.ArrayList" session = "true" contentType = "java/html" language = "java" errorPage ="errorpage.jsp" isErrorPage = "no"%>
<%@ include page= "anotherjsp.jsp"%>
<%@ taglib = .... %>
What are implicit objects in JSP?
request - instance of HttpSessionRequest, used as request.getAttribute and request.SetAttribute
Response- instance of HttpSessionResponse, used as response.get/setAttribute, response.SendRedirect(newurl)
Ssession - instance of HttpSession, used as session.get/setAttribute
out - instance of JspWriter,used as out.println()
config - instnace of ServletConfig, one config per one jsp page
application - instance of ServletContext, one context per application
page - JSP page, used as <@include page="anotherpage.jsp">
pageContext - Special object
