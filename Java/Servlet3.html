What is servlet chaining?
Output of one servlet is piped in to the 2nd servlet, output of 2nd servlet is piped to 3rd servlet and so on. the final servlet sends output to browser. this is called servlet chaining
What is listener
Whenever an event occurs, listener will be executed. Some examples of listerner are
ServletContextListener - when any parameter is added, removed or when the context is loaded this listener is invoked
ServletConfigListener HttpSessionListener etc.,
What is filter
Filters are like interceptors. Once implemented each request and response has to pass through the filter chain. One filter completes execution and its result is piped in to the next filter and so on, after the last filter, request/response reaches the specific party. Generally used for verification/validation of input data or logging/tracking of the response sent out from the servlets
