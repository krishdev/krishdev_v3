What bean scopes does Spring support? Explain them.

The Spring Framework supports following five scopes, three of which are available only if you use a web-aware ApplicationContext.

  • singleton: This scopes the bean definition to a single instance per Spring IoC container.

  • prototype: This scopes a single bean definition to have any number of object instances.

  • request: This scopes a bean definition to an HTTP request. Only valid in the context of a web-aware Spring ApplicationContext.

  • session: This scopes a bean definition to an HTTP session. Only valid in the context of a web-aware Spring ApplicationContext.

  • global-session: This scopes a bean definition to a global HTTP session. Only valid in the context of a web-aware Spring ApplicationContext.

What is Auto wiring?

The Spring container is able to autowire relationships between collaborating beans. This means that it is possible to automatically let Spring resolve collaborators (other beans) for your bean by inspecting the contents of the BeanFactory. 1 2 3

What are the important beans lifecycle methods?

There are two important bean lifecycle methods. The first one is setup which is called when the bean is loaded in to the container. The second method is the teardown method which is called when the bean is unloaded from the container.
